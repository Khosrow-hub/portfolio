Portfolio Website (Tailwind CSS)
A modern, multi-page developer portfolio built with Tailwind CSS v4 featuring custom animations, responsive design, and a complete project showcase.

Features

Multi-page Layout: 5 distinct pages (Home, About, Resume, Portfolio, Contact)

Tailwind CSS v4: Latest version with JIT compilation

Custom Animations: Typewriter effect, fade-ins, hover animations

Fully Responsive: Mobile-first design approach

Project Showcase: Highlights 4+ web applications

Contact Form: Interactive contact page

Technologies
HTML5

Tailwind CSS v4 (with build process)

Vanilla JavaScript

Custom CSS animations
